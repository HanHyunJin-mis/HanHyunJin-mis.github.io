recursion (재귀함수)

함수 호출 도중에 자기 자신을 다시 호출하는 것
base case(종료, 탈출, 조건) 필수

  1. 패턴을 찾고 점화식을 만든다!! (점화식 : 인접한 항들 사이의 관계식)
  2. 기저 조건을 만든다. (코드를 짜기 전에 기초가 되는 조건)

print는 I/O 작업이라 느리게 작동한다. (에러가 빨리 안남)


factorial (계승)

3! = 3 2 1
3! = 3 2!
2! = 2 1!

  1. 점화식
    fac(n) = fac(n-1)* n
  1. 기저 조건
    n == 1 or n == 0 return 1


fibonacci series (피보나치)

  1. 점화식
    fibo(n) -> n번째 수
    fibo(n-2) + fibo(n-1)
    fibo(n) = fibo(n-2) + fibo(n-1)
  1. 기저 조건
    if n ==1 then 0
    if n == 2 then 1
