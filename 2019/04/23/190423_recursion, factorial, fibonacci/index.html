factorial (계승)

3! = 3 2 1
3! = 3 2!
2! = 2 1!

  1. 점화식
    fac(n) = fac(n-1)* n
  1. 기저 조건
    n == 1 or n == 0 return 1


fibonacci series (피보나치)

  1. 점화식
    fibo(n) -> n번째 수
    fibo(n-2) + fibo(n-1)
    fibo(n) = fibo(n-2) + fibo(n-1)
  1. 기저 조건
    if n ==1 then 0
    if n == 2 then 1
